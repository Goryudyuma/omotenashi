# omotenashi

クローン
```
$ git clone https://github.com/kstm-su/omotenashi.git
```

依存関係を解決
```
$ npm install
```

bundle.jsを生成
```
$ npm run build
```

テスト用サーバーを建てる
```
$ npm start
```
